public void addArchiveResource(const QString & path, const QString & url, const QString & password, int priority)​

使用包含 Web 资源的本地存档 （.zip） 文件添加 url 映射项。这适用于随后创建的所有实例

参数​

  • path 本地存档文件的路径

  • password 存档的密码

  • priority 优先级
